🚀 Intelligent Automation

Upload a doc.
Ship tested code.
Zero effort.

DocXAI is an AI-powered pipeline on Google Cloud that reads your design document and automatically generates a Functional Specification, a Playwright test suite, and triggers a Cloud Build run — all from a single file upload.

01
📄
Upload Design Document
Drag & drop .txt, .md, .docx — stored in Google Cloud Storage
↓
02
🤖
AI Generates Functional Spec
Vertex AI (Gemini 2.5 Pro) analyses the document & writes a full spec
↓
03
📝
Spec Saved as .docx
OpenXML document saved back to Cloud Storage /output/
↓
04
🧪
Playwright Tests Generated
Gemini writes a full TypeScript test suite from the spec
↓
05
🏗️
Cloud Build Executes Tests
Playwright runs in Docker — results logged to Cloud Logging
